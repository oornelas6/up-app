# UP — Workout Tracker

A frictionless workout tracking app built for people who actually lift.
Zero typing. One-tap set logging. Streaks. PRs. Built on AWS.

## Live Demo
http://up-app-octavio.s3-website-us-east-1.amazonaws.com

## Stack
- AWS S3 — static hosting
- HTML, CSS, JavaScript

## Vision
UP is a mobile-first progressive overload tracker with game mechanics,
streak tracking, and a personality. Built to solve the friction problem
in every existing workout app.

## Status
v0.1 — Home screen prototype. Active development.
